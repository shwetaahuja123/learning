# Run

### In Bash
```
>cd bash
>./hello.sh
```

### In JS
```
>cd js
>npm i
>node Hello.js

//to run test
>npm test

//to check if current hostname is present in ../resources/known.hosts
>node findHostname.js

//to identify current env(python, node, etc)
>node checkEnv.js

//program that prints itself
>node printItself.js
```

### In Python
```
>cd python
>python Hello.py

//to run test
>python -m unittest test_hello.Testing

//to check if current hostname is present in ../resources/known.hosts
>python findHostname.py

//to identify current env(python, node, etc)
>python checkEnv.py

//program that prints itself
>python printItself.py
```
